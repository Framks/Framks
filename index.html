Projetos

Analise de Algoritmos de ordenamento

Projeto acadêmico no qual realizei a análise de desempenho de diferentes algoritmos de ordenação aplicados a listas encadeadas. A velocidade dos algoritmos foi medida em termos de tempo de execução, utilizando dados aleatórios de listas com tamanhos variando de 1.000 a 100.000 elementos, mantendo os mesmos dados para todos os algoritmos comparados.

  • Algoritmos: Bubble Sort, Merge Sort, Insertion Sort, Selection Sort, Radix Sort, Quick Sort

Gestão de Feedback Acadêmico (Atuando...)

O projeto "Gestão de Feedback Acadêmico" é uma aplicação web que facilita a coleta, o gerenciamento e a análise de feedbacks em ambientes acadêmicos, promovendo um ambiente de aprendizado colaborativo.

  • Registro e Autenticação de Usuários: Permite que professores e alunos criem contas e façam login no sistema.
  • Gerenciamento de Feedbacks: Envio e organização de feedbacks por alunos.
  • Visualização e Análise de Feedbacks: Professores visualizam feedbacks com filtros.

Tecnologias Utilizadas: React.js, Spring Boot, PostgreSQL, JWT.
